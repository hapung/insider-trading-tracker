# 📈 내부자 거래 추적기 (Insider Trading Tracker)

미국 주식 시장의 내부자(임원, 주요 주주) 거래 내역을 추적하고 분석하는 웹 애플리케이션입니다.

Finnhub API를 활용하여 SEC(미국 증권거래위원회)에 공시되는 Form 4 데이터를 수집하고, 사용자가 특정 기업의 내부자 거래 동향을 쉽게 파악할 수 있도록 돕는 것을 목표로 합니다.

---

## 🛠️ 사용 기술 (Tech Stack)

* **Backend:** Java, Spring Boot
* **Frontend:** React
* **Data Source:** Finnhub API
